Die Roller Pseudocode

Name: David Oberlander

Problem Statement

Write a program that will simulate the roll of an input number of dice. The solution should interact naratively with the user by first name, and randomly generated a number to simulate the roll of each single die requested. The output should show the random result for each separate die and then calculate the sum of the requested number of dice. Random numbers generated for each die must result in single digit integers between 1 and 6 inclusive.

1. Nouns and Verbs

Identify the nouns and verbs in the problem statement.

PROBLEM STATEMENT:
Write a program that will simulate the roll of an input number of dice. The solution should 
interact naratively with the user by first name, and randomly generated a number to simulate
the roll of each single die requested. The output should show the random result for each 
separate die and then calculate the sum of the requested number of dice. Random numbers 
generated for each die must result in single digit integers between 1 and 6 inclusive. 

Nouns: number of dice, first name, random generated number, sum, output, integers between 1 and 6

Verbs: roll, input, generate, request, show, calculate

2. Defining Diagram

Convert the nouns and verbs into variable names and processes and create a defining diagram.

INPUTS:
  numberOfDice
  firstName

PROCESSING:
  prompt user for numberOfDice
  convert numberOfDice to a number
  prompt user for firstName
  calculate randomly generated number between 1 and 6 inclusive for each numberOfDice
  display each generated number
  calculate the sum of the generated numbers
  display the calculated sum

OUTPUTS:
  generatedNumber for each die
  calculatedTotal

3. Solution Algorithm

Enter the Solution Algorithm below. 


RollDice
  Initialize strFirstName to ""
  Initialize strDieWord = "dice"
  Initialize strOutputString to ""
  Initialize intNumberDie to 0
  Initialize intGeneratedNumber to 0
  Initialize intSumOfDice to 0
  
  DO
    Prompt for strFirstName with "Please enter your first name:"
  WHILE strFirstName is not empty

  DO
    Prompt for intNumberDie with "Please enter the number of dice you want to use."
    Convert intNumberDie to a number
    IF intNumberDie is Not a Number OR intNumberDie is less than 0 OR (parseFloat(intNumberDie) does not equal parseInt(intNumberDie)) THEN
      Alert with "Please enter a positive number."
    ELSE IF intNumberDie is greater than 10 THEN
      Alert with "Sorry we only have 10 dice."
    ELSE IF intNumberDie is 1 THEN
      strDieWord = "die"
    END IF
  WHILE intNumberDie is Not a Number OR intNumberDie is less than 0 OR (parseFloat(intNumberDie) does not equal parseInt(intNumberDie)) OR intNumberDie is greater than 10

  Set strOutputString to "
Rolling " + intNumberDie + " " + strDieWord + "...
"

  IF intNumberDie is greater than zero THEN

    Set strOutputString to strOutputString + "
Hey " + strFirstName + ", you rolled a "

    LOOP starting at 1 up to (and including) intNumberDie in increments of 1
      Set intGeneratedNumber to Random number between 1 and 6
      Alert with "Die #" + increment + " is a " + intGeneratedNumber

      IF increment is not equal to intNumberDie OR intNumberDie is equal to 1 THEN
        IF intNumberDie is less than or equal to 2 THEN
          Set strOutputString to strOutputString + intGeneratedNumber + " "
        ELSE  
          Set strOutputString to strOutputString + intGeneratedNumber + ", "
        END IF
      ELSE 
        Set strOutputString to strOutputString + "and " + intGeneratedNumber
      END IF

      Set intSumOfDice to intSumOfDice + intGeneratedNumber
    ENDLOOP

    Set strOutputString to strOutputString + " for a grand total of " + intSumOfDice + ".
 
Hope you had fun rolling the dice!
"

  ELSE
      Set strOutputString to strOutputString + "
Hey " + strFirstName + ", you forgot the dice again. " + "Your grand total is " + intSumOfDice + "."
  END IF

  Display strOutputString to the "content" element's innerHTML

END
